Snake Game 
A simple and interactive Snake Game built using HTML, CSS, and JavaScript.
The game includes score tracking, high score saving using local storage, timer functionality, and restart game features.

🎮 Features
Classic Snake Game
Real-time score tracking
High score saved in browser localStorage
Game timer
Restart game option
Responsive game board
Smooth keyboard controls

🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
